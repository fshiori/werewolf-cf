Werewolf CF - 版本與技術資訊

📋 版本資訊

腳本名稱 Werewolf CF
版本 v1.0
原作 Pixmicat-based Werewolf Game（日文：Werewolf Online Game）
適應環境 Cloudflare Workers / Cloudflare Pages
授權 請參閱 GitHub 仓库

🛠️ 技術堆疊

本腳本使用以下 Cloudflare 服務建構：

☁️ Cloudflare Workers 🔗 Durable Objects 🗄️ D1（SQLite） 📦 R2 Storage ⚡ KV Namespace
Workers 處理 HTTP 路由、API 請求與遊戲邏輯
Durable Objects 管理每個遊戲房間的即時狀態與 WebSocket 連線
D1 SQLite 資料庫，儲存遊戲記錄與歷史資料
R2 物件儲存，存放玩家上傳的頭像圖示
KV 鍵值儲存，用於配置與暫存資料

📜 原始專案

本腳本基於 Pixmicat-based Werewolf Game 進行改寫與移植。
原始專案為 PHP + MySQL 架構的 BBS 型人狼遊戲。
本版本將後端完全改寫為 Cloudflare Workers，並重新設計前端介面。

⚠️ 已知限制

使用前請注意以下限制：
  • Cloudflare Workers 有 CPU 時間限制（免費版 10ms，付費版 30s/分鐘），大量運算可能觸及限制
  • Durable Objects WebSocket 連線有閒置超時機制
  • D1 資料庫有每日讀寫次數限制（免費版）
  • R2 儲存空間與操作次數在免費方案中有上限
  • 部分原版 PHP 專案的高階功能可能尚未完全移植
  • 前端為純靜態 HTML + JavaScript，無伺服器端渲染
